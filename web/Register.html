PlateletWatch Dengue Early Detection
Monitor Platelets. Detect Risks Early.

A microscope-integrated web application that supports early dengue detection through platelet count analysis from blood samples.

  • Real-time platelet count tracking
  • Automated dengue risk stratification
  • AI-assisted blood analysis
  • Secure patient data management
⚠️ Disclaimer: This system is intended for early screening and detection purposes only. It does not replace professional medical diagnosis, treatment, or consultation.
